Description
令 G=(V,E)是一个具有n个结点的无向图。如果一个结点的子集中没有两个结点交于一条边，这个子集就叫独立集。
一般来说，找最大独立集是很困难的（NP难）。但是如果图G足够“简单”，那么我们可以有效的完成这个任务。

如果图 G=(V,E)的结点可以写作v1,v2,...,vn，在vi和vj之间有一条边当且仅当数字i和j恰好差1，那么把这个图
叫做“一条路径图”。每个结点都有权wi与之相关。一条路径图像这样，
<left><img src="pic/17102_1446109934379_4613.jpg"></left>

上图画的五个结点的路径，权为结点内标的整数。
这里最大权值和的独立集为v2和v5，权值和为w2+w5=8+6=14
这里定义当独立集为空时，权值和为负无穷大。也就是说，只要这个图有结点，独立集结点数至少有1个。

给出一个算法，计算含有n个结点的路径G的一个总权最大的独立集。
并分析算法的运行时间是n的多项式，与权值无关。

输入格式
第一行：一条路径图的节点个数n  (n<=100000)
第二行：每个结点具有的权值w1  w2  ...  wn   （权值为整数，可能有正、0、也可能有负）

输出格式
独立集的最大总权值。

输入样例
5
1 8 6 3 6

输出样例
14

解释：独立集为v2和v5，总权值为w2+w5=8+6=14

提示
求一般图的“最大独立集”问题是NP完全问题。所幸此题是特殊的图，可以高效求解。

此题比较简单。
假设从最左边结点一直沿着一条路径图到结点vi为止，具有的独立集最大权值之和为f(i)。
f(1) = w1;
f(2) = max{w1,w2};
f(i) = max{f(i-1), f(i-2)+wi, wi};     2<i<=n

由于这里权值有正有负，所以第i个结点是否加入独立集，可以存在如下三种状态：
（1）前i-1个结点选最大权值和的独立集，第i个结点不加入；
（2）前i-2个结点选独立集，第i个结点加入，因为第i个结点和前i-2个选出的独立集中
任何一个结点都不会相连；
（3）由于前i-2个结点选独立集的最大权值和可能为负，还不如第i个结点单独一个作为
新独立集，因此只选第i个结点；

最后的，f(n)即为原问题所求。
