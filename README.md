## 目录结构

```
{单元目录}/
├── {题号} {英文题目名}/
│   ├── main.cpp              ← 唯一源文件
│   └── problem.png           ← 仅题目描述为图片时
├── {题号} {英文题目名}.txt    ← 与 main.cpp 内容相同
└── ...
```

## 构建与运行

每个 `main.cpp` 是独立程序：

```bash
g++ -std=c++17 -O2 -o a.exe "1General Methods/9715 Max Rectangle Area/main.cpp"
./a.exe < input.txt
```

## 代码规范

- 头文件白名单（14 种），`using namespace std;`
- 代码在上，OJ 题目描述原文以 `/* */` 块注释放在末尾
- 首行 `//{题号} {中文题目名}`
- 不写作者/日期头部
- 详细规范见 [`docs/代码规范.md`](docs/代码规范.md)
